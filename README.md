# YiYue GitHub Profile Kit

这是一套为 YiYue 准备的 GitHub 个人品牌文件，风格方向是：

> AI × Blockchain Finance × Personal Productivity

## 文件说明

- `docs/BRAND_POSITIONING.md`：个人品牌定位和页面结构方案。
- `profile-repo/README.md`：GitHub 个人主页 README。
- `resume-repo/README.md`：公开版个人简历 README。
- `ai-agents-lab/README.md`：AI agents 实验室仓库首页。
- `learning-roadmap/README.md`：学习路线仓库首页。
- `github-pages-site/`：GitHub Pages 个人网站。
- `docs/BANNER_PROMPTS.md`：头像、横幅、项目图的 AI 生成提示词。
- `docs/SETUP_STEPS.md`：小白上传 GitHub 的步骤。

## 使用前必须替换

- `YOUR_GITHUB_USERNAME`
- `your-email@example.com`
- `your-wechat-id`
- `https://x.com/your_handle`
- `https://t.me/your_handle`

## 建议下一步

1. 先确定 GitHub 用户名。
2. 创建同名 Profile 仓库。
3. 上传 `profile-repo/README.md`。
4. 创建 `yiyue-resume`、`ai-agents-lab`、`learning-roadmap`。
5. 创建 `YOUR_GITHUB_USERNAME.github.io` 并上传网站文件。
6. 用 `docs/BANNER_PROMPTS.md` 生成头像和横幅。
